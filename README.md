# What is this

This small script get the rating (standard, rapid, blitz) of top 100 players in Australia and save it to an Excel file.

# How to run it
```
python -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```

Install https://github.com/xRuiAlves/fide-ratings-scraper
